# antenna-xphase-case
3D-printable (FDM, ASA) enclosure for a PA0MBJ-based X-Phase QRM eliminator, with mast-mount support and provisions for RF circuitry used in phase noise cancellation.


Custom enclosure for a PA0MBJ-based X-Phase QRM eliminator. Designed to house the RF circuitry that cancels local interference using phase-shift techniques with a main and auxiliary antenna, improving signal clarity in noisy environments. The enclosure includes a mounting system compatible with mast diameters: 15mm, 36mm, 50mm. It is optimized for FDM 3D printing using ASA material, providing good UV resistance and durability for outdoor use.
